FarmChain Assist: Voice-First Decentralized Agriculture

FarmChain Assist is a high-performance platform designed to eliminate literacy barriers in the agricultural supply chain using Voice-First AI and Blockchain technology. It replaces complex data entry with a "Zero-Typing" interface for rural farmers.

Core Functionality
Voice-Enabled Authentication: Secure login using Aadhaar ID captured via Web Speech API audio input, mapped to Supabase Auth.
On-Chain Provenance: Harvest data is minted on the Polygon Blockchain, creating an immutable "Digital Passport" for every crop batch.
Multilingual Assistant: Integrated Text-to-Speech provides real-time feedback in Kannada, Hindi, and English to guide non-technical users.
Smart Logistics: QR-based digital handshakes ensure a transparent chain of custody from farm to consumer.

Technical Stack
Verification & Trust Model
Identity: Aadhaar-linked profiles prevent fraudulent account creation.
Location: GPS Geo-fencing validates that crop registration occurs on verified farmland.
Integrity: Smart Contracts on Polygon ensure that pricing and origin data cannot be altered by middlemen.

Installation
Clone the repository.

Install dependencies: bun install

Launch development server: bun dev

Project Goal
To provide a secure, accessible, and transparent marketplace where the Aadhaar-verified identity of a farmer is cryptographically linked to the quality of their harvest.
